Vapor is an open-source HTTP web framework written in Swift. Built on top of SwiftNIO, it gives you a fast, expressive, and type-safe foundation for building server-side applications — from REST APIs to full-featured web services.

## What you can build

Vapor is designed for server-side Swift projects of all kinds:

- **REST APIs** — Expressive routing, Codable content encoding, and full HTTP/2 support
- **Web applications** — Sessions, views, file serving, and WebSocket support
- **Microservices** — Lightweight, async-first, and deployable via Docker or Linux

## Key features

  <Card title="Async/Await" icon="bolt">
